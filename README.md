# ListBee Docs

Developer documentation for [docs.listbee.so](https://docs.listbee.so), powered by [Fern](https://buildwithfern.com).

## Development

```bash
# Sync OpenAPI spec
make sync-openapi

# Preview locally
make preview

# Build
make build
```

## Structure

- **Documentation tab** — Hand-written MDX guides
- **API Reference tab** — Auto-generated from OpenAPI spec
- **Changelog tab** — Release notes
