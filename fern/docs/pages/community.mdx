ListBee is built in the open. The API client libraries and n8n node are open source. File issues, send PRs, or just follow along.

---

## GitHub

All public repos live under the `listbee-dev` org:

- **GitHub org:** [github.com/listbee-dev](https://github.com/listbee-dev)
- **Python SDK:** [github.com/listbee-dev/listbee-python](https://github.com/listbee-dev/listbee-python)
- **n8n community node:** [github.com/listbee-dev/n8n-nodes-listbee](https://github.com/listbee-dev/n8n-nodes-listbee)

File bugs and feature requests as GitHub issues in the relevant repo.

---

## Email

For questions not suited to a public issue tracker — billing, account access, security — email:

## Security

To report a security vulnerability, email damian@listbee.so with subject line `[SECURITY]`. Do not file security issues publicly on GitHub.
