A user says "sell my ebook for $19." The agent hosts the file somewhere (S3, Drive, its own server), creates a ListBee listing pointing at it, and publishes. Two API calls.

## Full script

```python
import os
from listbee import ListBee, Deliverable

client = ListBee(api_key=os.environ["LISTBEE_API_KEY"])

# 1. Create a published static listing with the deliverable baked in
created = client.listings.create(
    name="Python Automation Handbook",
    price=1900,                                  # $19.00
    description="A practical guide to automating repetitive tasks with Python.",
    fulfillment_mode="static",
    deliverable=Deliverable.url("https://cdn.example.com/handbooks/python-automation.pdf"),
)

listing = created.listing
signing_secret = created.signing_secret          # store this — webhook HMAC key

print(f"Listing created: {listing.id} (status={listing.status})")

# 2. Publish
published = client.listings.publish(listing_id=listing.id)

print(f"Status: {published.status}")             # published
print(f"Product page: {published.url}")          # https://buy.listbee.so/pY2xK8m
print(f"Sellable: {published.readiness.sellable}")  # True, once Stripe is connected
```

## What happens

**Step 1 — Create.** The listing is created in `draft` status with `fulfillment_mode=static` and a URL deliverable. The response wraps the listing and a one-time `signing_secret` for webhook verification.

**Step 2 — Publish.** `POST /v1/listings/{id}/publish` moves the listing to `published`. The product page is immediately live. If the account has Stripe Connect set up, the buy button is active; if not, the page renders with a disabled button until onboarding completes (see [Readiness](/readiness)).

When a buyer pays, ListBee auto-delivers the URL — the buyer is redirected to it from the unlock page. No extra agent code runs.

## Reading the readiness object

```json
{
  "object": "listing",
  "id": "lst_abc123",
  "status": "draft",
  "readiness": {
    "sellable": false,
    "actions": [
      {
        "code": "listing_unpublished",
        "kind": "api",
        "priority": "required",
        "message": "Publish the listing to make it buyable.",
        "resolve": {"method": "POST", "endpoint": "/v1/listings/lst_abc123/publish"}
      }
    ],
    "next": "listing_unpublished"
  }
}
```

`readiness.next` points to the single highest-priority action. Agents resolve it, refetch, and the list converges to `sellable: true`.

## Variations

**Text deliverable** — deliver a license key or block of Markdown:

```python
client.listings.create(
    name="Premium License",
    price=4900,
    fulfillment_mode="static",
    deliverable=Deliverable.text("Your license key: XXXX-XXXX-XXXX-XXXX"),
)
```

**AI-generated content per order** — use `fulfillment_mode=async` with an `agent_callback_url`. On `order.paid`, your server generates content and calls `POST /v1/orders/{id}/fulfill` to push it. See [Fulfillment](/fulfillment).

```python
client.listings.create(
    name="Personalized Market Report",
    price=9900,
    fulfillment_mode="async",
    agent_callback_url="https://your-agent.example.com/hooks/listbee",
    metadata={"report_type": "market_analysis"},
)
```
