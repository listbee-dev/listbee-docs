---
title: Selling physical products
description: Use external fulfillment with address collection to sell physical goods through ListBee.
---

ListBee handles payment. You handle shipping. Use external fulfillment with a checkout schema to collect the buyer's shipping address, then ship the order and update its status via the API.

---

## How it works

1. Create a listing with `fulfillment: "external"` and a `checkout_schema` that includes an address field.
2. Buyer visits the product page, fills in their shipping address, and pays.
3. ListBee fires an `order.paid` webhook with the buyer's address in `checkout_data`.
4. You ship the product and call `POST /v1/orders/{id}/ship` with tracking info.
5. ListBee fires an `order.shipped` webhook.
6. When the buyer receives the product, call `POST /v1/orders/{id}/fulfill` to mark it complete.

## Walkthrough: selling shoes

### Step 1: Create the listing

<CodeGroup>

```python title="Python"
import httpx

resp = httpx.post(
    "https://api.listbee.so/v1/listings",
    headers={"Authorization": "Bearer lb_..."},
    json={
        "name": "Handmade Leather Boots",
        "price": 18900,
        "fulfillment": "external",
        "checkout_schema": [
            {
                "key": "size",
                "type": "select",
                "label": "Shoe size",
                "required": True,
                "options": ["EU 38", "EU 39", "EU 40", "EU 41", "EU 42", "EU 43", "EU 44"],
            },
            {
                "key": "shipping_address",
                "type": "address",
                "label": "Shipping address",
                "required": True,
            },
            {
                "key": "gift_note",
                "type": "text",
                "label": "Gift note (optional)",
                "required": False,
            },
        ],
    },
)
listing = resp.json()

print(listing["fulfillment"])  # external
print(listing["url"])          # https://buy.listbee.so/r7kq2xy9
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee({ apiKey: "lb_..." });

const listing = await client.listings.create({
  name: "Handmade Leather Boots",
  price: 18900,
  fulfillment: "external",
  checkoutSchema: [
    {
      key: "size",
      type: "select",
      label: "Shoe size",
      required: true,
      options: ["EU 38", "EU 39", "EU 40", "EU 41", "EU 42", "EU 43", "EU 44"],
    },
    {
      key: "shipping_address",
      type: "address",
      label: "Shipping address",
      required: true,
    },
    {
      key: "gift_note",
      type: "text",
      label: "Gift note (optional)",
      required: false,
    },
  ],
});

console.log(listing.fulfillment);  // external
console.log(listing.url);          // https://buy.listbee.so/r7kq2xy9
```

</CodeGroup>

### Step 2: Receive the order webhook

When a buyer pays, ListBee sends an `order.paid` event to your webhook endpoint. The order includes `checkout_data` with the buyer's responses and `shipping_address` parsed from the address field.

```json
{
  "id": "evt_...",
  "object": "event",
  "type": "order.paid",
  "created_at": "2026-04-02T10:00:00Z",
  "data": {
    "object": {
      "object": "order",
      "id": "ord_abc123",
      "status": "paid",
      "amount": 18900,
      "currency": "usd",
      "listing_id": "lst_...",
      "listing_slug": "handmade-leather-boots",
      "buyer_email": "buyer@example.com",
      "checkout_data": {
        "size": "EU 42",
        "gift_note": "Happy birthday!"
      },
      "shipping_address": {
        "line1": "123 Main St",
        "line2": "Apt 4B",
        "city": "Portland",
        "state": "OR",
        "postal_code": "97201",
        "country": "US"
      },
      "fulfillment_status": "pending",
      "paid_at": "2026-04-02T10:00:00Z"
    }
  }
}
```

### Step 3: Ship the order

After you ship, call `POST /v1/orders/{id}/ship` with tracking info:

<CodeGroup>

```python title="Python"
import httpx

resp = httpx.post(
    "https://api.listbee.so/v1/orders/ord_abc123/ship",
    headers={"Authorization": "Bearer lb_..."},
    json={
        "carrier": "USPS",
        "tracking_code": "9400111899223456789012",
        "seller_note": "Ships from Portland, OR. Estimated delivery: 3-5 days.",
    },
)
order = resp.json()

print(order["status"])              # paid
print(order["fulfillment_status"])  # shipped
print(order["carrier"])             # USPS
print(order["tracking_code"])       # 9400111899223456789012
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee({ apiKey: "lb_..." });

const order = await client.orders.ship("ord_abc123", {
  carrier: "USPS",
  trackingCode: "9400111899223456789012",
  sellerNote: "Ships from Portland, OR. Estimated delivery: 3-5 days.",
});

console.log(order.status);              // paid
console.log(order.fulfillment_status);  // shipped
console.log(order.carrier);             // USPS
console.log(order.tracking_code);       // 9400111899223456789012
```

</CodeGroup>

This fires an `order.shipped` webhook event.

### Step 4: Mark as fulfilled

When the buyer receives the product:

<CodeGroup>

```python title="Python"
import httpx

resp = httpx.post(
    "https://api.listbee.so/v1/orders/ord_abc123/fulfill",
    headers={"Authorization": "Bearer lb_..."},
    json={},
)
order = resp.json()

print(order["status"])              # fulfilled
print(order["fulfillment_status"])  # fulfilled
print(order["fulfilled_at"])        # 2026-04-05T14:30:00Z
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee({ apiKey: "lb_..." });

const order = await client.orders.fulfill("ord_abc123", {});

console.log(order.status);              // fulfilled
console.log(order.fulfillment_status);  // fulfilled
console.log(order.fulfilled_at);        // 2026-04-05T14:30:00Z
```

</CodeGroup>

## Order lifecycle for physical products

```
PENDING → PAID → (shipped) → FULFILLED
```

| Status | `fulfillment_status` | Meaning |
|--------|---------------------|---------|
| `pending` | `null` | Buyer opened checkout, hasn't paid |
| `paid` | `pending` | Payment confirmed, waiting for shipment |
| `paid` | `shipped` | Package shipped, tracking info available |
| `fulfilled` | `fulfilled` | Buyer received the product |
