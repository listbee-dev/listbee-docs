## Step 1: Create an account

No console signup needed. The agent bootstraps directly via the API.

<CodeBlocks>
<CodeBlock title="Python">
```python
from listbee import ListBee

# Send OTP to the owner's email
session = ListBee.bootstrap.start(email="seller@example.com")

# Ask the human for the 6-digit code from their email
code = input("Enter the code: ")

# Verify and complete bootstrap
ListBee.bootstrap.verify(session=session.session, code=code)
result = ListBee.bootstrap.complete(session=session.session)

api_key = result.api_key      # lb_... — save this
account_id = result.account_id  # acc_...
```
</CodeBlock>
<CodeBlock title="curl">
```bash
# 1. Send OTP
curl -X POST https://api.listbee.so/v1/bootstrap \
  -H "Content-Type: application/json" \
  -d '{"email": "seller@example.com"}'
# → {"session": "sess_...", "otp_sent": true}

# 2. Verify (ask human for the code)
curl -X POST https://api.listbee.so/v1/bootstrap/verify \
  -H "Content-Type: application/json" \
  -d '{"session": "sess_...", "code": "847291"}'

# 3. Complete bootstrap
curl -X POST https://api.listbee.so/v1/bootstrap/complete \
  -H "Content-Type: application/json" \
  -d '{"session_id": "sess_..."}'
# → {"object": "bootstrap", "account_id": "acc_...", "api_key": "lb_..."}
```
</CodeBlock>
</CodeBlocks>

Use the `api_key` from the response as `Authorization: Bearer lb_...` for all subsequent requests.

## Step 2: Create a listing

A listing is a sellable item with a hosted product page. Required fields: `name` and `price` (in cents).

<CodeBlocks>
<CodeBlock title="Python">
```python
client = ListBee(api_key=api_key)

listing = client.listings.create(
    name="SEO Playbook 2026",
    price=1900,  # $19.00
)

print(listing.id)          # lst_7kQ2xY9mN3pR5tW1vB8a
print(listing.short_code)  # aB3xK9m
print(listing.url)         # https://buy.listbee.so/aB3xK9m
```
</CodeBlock>
<CodeBlock title="curl">
```bash
curl -X POST https://api.listbee.so/v1/listings \
  -H "Authorization: Bearer lb_..." \
  -H "Content-Type: application/json" \
  -d '{"name": "SEO Playbook 2026", "price": 1900}'
```
</CodeBlock>
</CodeBlocks>

## Step 3: Attach a deliverable and publish

Attach the content buyers receive after paying — a URL, file, or text. Then publish.

<CodeBlocks>
<CodeBlock title="Python">
```python
# Attach a deliverable
client.listings.add_deliverable(
    listing_id=listing.id,
    type="url",
    value="https://example.com/seo-playbook-download",
)

# Publish — product page goes live immediately
published = client.listings.publish(listing_id=listing.id)
print(published.status)  # published
print(published.url)     # share this with buyers
```
</CodeBlock>
<CodeBlock title="curl">
```bash
# Attach a deliverable
curl -X POST https://api.listbee.so/v1/listings/lst_.../deliverables \
  -H "Authorization: Bearer lb_..." \
  -H "Content-Type: application/json" \
  -d '{"type": "url", "value": "https://example.com/seo-playbook-download"}'

# Publish
curl -X POST https://api.listbee.so/v1/listings/lst_.../publish \
  -H "Authorization: Bearer lb_..."
```
</CodeBlock>
</CodeBlocks>

The product page is live. If Stripe is not connected yet, the page renders but the buy button is disabled — it shows "Coming soon" until the human connects Stripe.

## Step 4: Connect Stripe (when ready to sell)

Check account readiness to see what's needed:

<CodeBlocks>
<CodeBlock title="Python">
```python
account = client.account.get()
if not account.readiness.sellable:
    # Present the Stripe URL to the human
    action = account.readiness.actions[0]
    print(f"Connect Stripe: {action.resolve.url}")
```
</CodeBlock>
<CodeBlock title="curl">
```bash
curl https://api.listbee.so/v1/account \
  -H "Authorization: Bearer lb_..."
```
</CodeBlock>
</CodeBlocks>

The human opens the Stripe Connect URL, completes onboarding (5 minutes), and the listing's buy button activates automatically.

## What happens next

When a buyer visits the product page and pays:
1. ListBee creates a Stripe Checkout session on the seller's connected account
2. After payment, the order is created with status `paid`
3. If the listing has deliverables, ListBee auto-delivers and emails the buyer
4. `order.paid` and `order.fulfilled` webhook events fire

Set up a [webhook](/webhooks) to get notified of orders programmatically.
