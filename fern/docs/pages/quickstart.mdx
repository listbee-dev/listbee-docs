Three steps to go from zero to a live listing: create an account, connect Stripe, create a listing.

---

## Step 1: Create an account

Send your email to receive a one-time code. Then verify the code to get your API key.

<CodeGroup>

```python title="Python"
import httpx

BASE = "https://api.listbee.so"

# 1a. Request OTP
httpx.post(f"{BASE}/v1/account/signup", json={"email": "you@example.com"})

# Check your inbox for the 6-digit code, then:

# 1b. Verify and get API key
resp = httpx.post(
    f"{BASE}/v1/account/verify",
    json={"email": "you@example.com", "code": "123456"},
)
data = resp.json()

api_key = data["api_key"]           # lb_...
account = data["account"]           # account object
print(f"API key: {api_key}")
print(f"Account: {account['id']}")  # acc_...
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee(); // no apiKey needed for signup

// 1a. Request OTP
await client.signup.create({ email: "you@example.com" });

// Check your inbox for the 6-digit code, then:

// 1b. Verify and get API key
const result = await client.signup.verify({
  email: "you@example.com",
  code: "123456",
});

const apiKey = result.api_key;        // lb_...
const account = result.account;       // account object
console.log(`API key: ${apiKey}`);
console.log(`Account: ${account.id}`); // acc_...
```

</CodeGroup>

Store the API key — all subsequent requests use it as a Bearer token.

## Step 2: Connect Stripe

Submit your Stripe secret key. ListBee stores it encrypted and uses it to process payments.

<CodeGroup>

```python title="Python"
import httpx

BASE = "https://api.listbee.so"
API_KEY = "lb_..."  # from step 1

resp = httpx.post(
    f"{BASE}/v1/account/stripe-key",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={"secret_key": "sk_live_..."},
)
account = resp.json()

print(account["readiness"]["operational"])  # True — ready to sell
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee({ apiKey: "lb_..." }); // from step 1

const account = await client.stripe.setKey({
  secretKey: "sk_live_...",
});

console.log(account.readiness.operational); // true — ready to sell
```

</CodeGroup>

`readiness.operational` is `true` when Stripe is connected and the account can process payments.

<Note>
Don't have a Stripe account yet? Call `POST /v1/account/stripe/connect` to start a Stripe Connect onboarding flow. You'll get back a URL to complete setup in your browser.
</Note>

## Step 3: Create a listing

Three required fields: `name`, `price` (in cents), and `content` (a URL or text payload). Currency is determined by your account settings.

<CodeGroup>

```python title="Python"
import httpx

BASE = "https://api.listbee.so"
API_KEY = "lb_..."  # from step 1

resp = httpx.post(
    f"{BASE}/v1/listings",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "name": "Python Async Patterns",
        "price": 1900,      # $19.00 in cents
        "content": "https://your-bucket.s3.amazonaws.com/python-async-patterns.pdf",
    },
)
listing = resp.json()

print(listing["url"])                      # https://buy.listbee.so/r7kq2xy9
print(listing["readiness"]["sellable"])    # True — buyers can purchase now
```

```typescript title="TypeScript"
import { ListBee } from "listbee";

const client = new ListBee({ apiKey: "lb_..." }); // from step 1

const listing = await client.listings.create({
  name: "Python Async Patterns",
  price: 1900,     // $19.00 in cents
  content: "https://your-bucket.s3.amazonaws.com/python-async-patterns.pdf",
});

console.log(listing.url);                    // https://buy.listbee.so/r7kq2xy9
console.log(listing.readiness.sellable);     // true — buyers can purchase now
```

</CodeGroup>

The `url` is the product page. Share it with buyers. When `readiness.sellable` is `true`, the listing is live.

If `sellable` is `false`, check `readiness.actions` — each action has a `code`, `kind` (`api` or `human`), and a `resolve` object with the exact endpoint or URL to fix it.

## Copy for AI assistants

```python
# ListBee quickstart — three steps from zero to a live listing
#
# Step 1: Account
#   POST /v1/account/signup    { "email": "..." }
#     → 200 { object, email, status, message }   (OTP sent)
#   POST /v1/account/verify    { "email": "...", "code": "123456" }
#     → 201 { object, account, api_key }
#
# Step 2: Stripe
#   POST /v1/account/stripe-key  { "secret_key": "sk_live_..." }
#     → 200 account object with readiness.operational: true
#   (New Stripe users: POST /v1/account/stripe/connect → { url } to open in browser)
#
# Step 3: Listing
#   POST /v1/listings  { "name", "price" (cents), "content" }
#     → 201 { object, id, slug, url, readiness: { sellable, actions, next }, ... }
#
# All requests: Authorization: Bearer lb_...
# Errors:       RFC 9457 { type, title, status, detail, code, param }
# Docs:         https://docs.listbee.so
# Spec:         https://api.listbee.so/openapi.json
```
